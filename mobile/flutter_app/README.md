# UniEventAI - Flutter App

Bu klasör Flutter/Dart ile geliştireceğimiz mobil uygulama için başlangıç dosyalarını içerir.

Başlangıç:
1. Flutter SDK kurun: https://flutter.dev
2. IDE (VS Code / Android Studio) kurun ve Flutter eklentilerini yükleyin.
3. Bu klasörde `flutter create .` komutunu çalıştırarak temel proje iskeletini oluşturabilirsiniz.

Not: Bu repo yalnızca başlangıç ve yönergeler içerir; gerçek uygulama kodu `lib/` altındadır.
