# UniEventAI Backend (Node.js)

Basit bir Express API içeren başlangıç dizinidir.

Hızlı başlatma:
1. Node.js 18+ kurun
2. `cd backend/nodejs` ve `npm install`
3. `.env` dosyasını oluşturun (`.env.example`'i referans alın)
4. `npm start` veya geliştirme için `npm run dev`

Not: Gerçek veritabanı bağlantısı için `MONGO_URI` çevresel değişkenini ayarlayın.
